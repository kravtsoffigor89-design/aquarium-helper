# Aquarium Helper

A desktop Java application built with Swing GUI that calculates the real water volume of an aquarium and evaluates biological compatibility and density for specific fish species.

## Features
* Interactive GUI for dimension inputs (length, width, height).
* Automatic calculation of gross and net water volume (with a 10% safety margin for substrate and air space).
* Fish stocking density validator based on biological requirements.
* Compatibility guidelines based on fish temperament.

## Technologies Used
* Java 17
* Java Swing & AWT (Desktop UI Framework)
